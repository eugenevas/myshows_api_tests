

Краткие пояснения:
Фикстура add_test_series подключается к БД напрямую через psycopg, очищает таблицу series, добавляет три записи с нужными статусами (переведёнными в базовое значение).
После теста таблица очищается.
Тест test_get_series делает запрос к API, получает JSON-список, сравнивает полученные сериалы с теми, что были добавлены.
Важный момент — статус в API и в базе различается, поэтому при добавлении и проверке применяется словарь API_SERIES_STATUS_TO_DB_MAPPING.
Предполагается, что API возвращает статус в формате API (т.е. на русском, как в expected_series).
